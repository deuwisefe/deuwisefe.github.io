ES Module 环境 + ArcGIS 的三种最简化配置

分析 ES Module 结合 ArcGIS 三种方式的的特点与优劣

方法一：配置 webpack 加载方式

特点：可以使用 ES Module 的方式加载 GIS 模块，但是最终使用的还是外部资源

对应项目：前端基础框架，GIS 前端框架，广西水利，预警云

思路如下：

  1. libraryTarget 控制 webpack 打包的内容暴露为 amd 的形式
  2. externals 防止将 ArcGIS 的包打包到 bundle 中，而是在运行时再去从外部获取这些扩展依赖
  3. index.html 中引入外部 dojo，用它的 AMD 当加载器，加载业务模块和 GIS 模块
  4. 用 ejs 模板渲染 entry

webpack.config.js

const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {
  entry: { index: './src/index.js' },
  output: {
    libraryTarget: 'amd'
  },
  externals: (context, request, callback) => {
    if (/^(dojo|dojox|dijit|esri)/.test(request)) {
      return callback(null, `amd ${request}`);
    }
    return callback();
  },
  plugins: [
    new HtmlWebpackPluginj({
      chunks: ['index'],
      inject: false,
      template: './index.html'
    })
  ]
};

方法二：使用@arcgis/webpack-plugin

特点：可以使用 ES Module 的方式加载 GIS 模块，且 ArcGIS 的相关文件都在本地

对应项目：暂无

备注：官方暂时的解决方案，坑多。如果需要用于项目开发的话，webpack 需要配置的地方较多，文件提交，加载速度，IE 兼容性等。

思路如下：

  1. 使用 ArcGISPlugin
  2. 由于是本地使用，需要配置相关 loader（图片，字体，babel 等）
  3. 配置 HtmlWebPackPlugin 的 chunksSortMode:none
  4. 配置 node 环境配置

webpack.config.js

const ArcGISPlugin = require('@arcgis/webpack-plugin');
const HtmlWebPackPlugin = require('html-webpack-plugin');
const path = require('path');
const webpack = require('webpack');
module.exports = function(_, arg) {
  const config = {
    entry: {
      index: ['./src/index.js']
    },
    output: {
      filename: '[name].[chunkhash].js',
      publicPath: ''
    },
    module: {
      rules: [
        { exclude: /node_modules/, test: /\.js?$/ },
        { test: /\.(jpe?g|png|gif|svg|webp)$/, loader: 'url-loader' },
        {
          test: /\.(wsv|ttf|otf|eot|woff(2)?)(\?[a-z0-9]+)?$/,
          use: [{ loader: 'file-loader' }]
        },
        { test: /\.css/, use: ['style-loader', 'css-loader'] }
      ]
    },
    plugins: [
      new ArcGISPlugin({ useDefaultAssetLoaders: false }),
      new HtmlWebPackPlugin({
        template: './public/index.html',
        filename: './index.html',
        favicon: './public/assets/favicon.ico',
        chunksSortMode: 'none'
      })
    ],
    resolve: { extensions: ['.js', '.css'] },
    node: { process: false, global: false, fs: 'empty' }
  };
  return config;
};

方法三：使用 esri-loader

特点：加载方式不同，需要使用 esri-loader 提供的加载器加载，优点是可以方便的集成到任意框架里面

对应项目：A2

参考文档

使用方式

import { loadModules } from 'esri-loader';

loadModules(['esri/views/MapView', 'esri/WebMap'])
  .then(([MapView, WebMap]) => {
    var webmap = new WebMap({
      portalItem: {
        id: 'f2e9b762544945f390ca4ac3671cfa72'
      }
    });
    var view = new MapView({
      map: webmap,
      container: 'viewDiv'
    });
  })
  .catch(err => {
    console.error(err);
  });
